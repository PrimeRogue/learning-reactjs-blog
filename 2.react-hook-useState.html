2. React hook useState

1. Trường hợp initState là một mảng hoặc một Object

Nên làm theo các bước sau:

            
const [todoList, setTodoList] =useState(["Todo1", "Todo2"])
function handleSomething(){
   // 1. clone
   const newTodoList = [...todoList]

   // 2. do something

   // 3. setState
   setTodoList(newTodoList)
}

2. useState dùng REPLACING chứ không dùng MERGING

setState trong class component thì dùng MERGING

            
this.state = {name: "A", color: "brown"}; 
this.setState({color:'green'});
// result: {name: "A", color: "green"}

useState trong functional component thì dùng REPLACING

            
const [person, setPerson] = useState({name: "A", color: "brown"});
setPerson({color: "green"});
// result: {color: "green"}

Giải quyết MERGING trong useState

            
const [person, setPerson] = useState({name: "A", color: "brown"});
setPerson({...person, color: "green"});
// result: {name: "A", color: "green"}
