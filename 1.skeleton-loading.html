1. Skeleton Loading

1. CSS

Viết styling cho class skeleton.

            
.skeleton {
    background-color: #35363a;
    background-image: linear-gradient(110deg, #ececec 8%, #f5f5f5 18%, #ececec 33%);
    background-size: 200% 100%;
    animation: 1.5s shine linear infinite;
}

@keyframes shine {
    to {
        background-position-x: -200%;
    }
}

Thêm class skeleton vào các thành phần cần tạo skeleton loading. Styling width, height, border-radius cho các thành phần. 

            
.tour .image.skeleton {
    width: 100%;
    height: 20rem;
}
    
.tour h3.skeleton {
    height: 1.6rem;
    width: 100%;
    border-radius: 0.25rem;
}

2. Component Loading

Viết component Loading có cấu trúc tương tự như component cần Loading Skeleton (ở ví dụ dưới là component Tour) với data là rỗng. 

            
const Loading = ({ tours }) => {
    return (
        <div className="tour-container">
        {tours.map((item) => {
            return (
            <div key={item.id} className="tour">
                <div className="top">
                    <div className="image skeleton"></div>
                </div>
                <div className="bottom">
                    <h3 className="skeleton"></h3>
                    <p className="skeleton"></p>
                </div>
            </div>
            );
        })}
        </div>
    );
};
