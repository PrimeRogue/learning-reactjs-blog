1. Props

Props là 1 object, giá trị của props có thể là 1 hàm

           
const props = {
  srcImage: "https://source...",
  srcAvatar: "https://source...",
  title: "Learning ReactJS Framework",
  author: "Bùi Trần Thiên Ân",
  onClick : () => true
}

Sử dụng props

        
// Truyền props làm đối số trong component
function YoutubeItem(props) {
  return (
    <div>
      <img src={props.srcImage} alt="" />
    </div>
  );
}

// Sử dụng props trong component
<YoutubeItem srcImage="https://source.unsplash.com/random">
</YoutubeItem>

1. Truyền vào component

            <List people={data} />

Trong đó, data là 1 mảng. Khi đó props sẽ chứa

            
const props ={
  people: [...]
}

Có thể sử dụng destructoring, khi đó


const {people} = props

2. Sử dụng props trong hàm component, có thể thay props thành destructoring ở trên 

            
const List = (props) => {
return (
  <div>
    {props.people.map((item) => {
      const { id, name, age, image } = item;
      return (
        <div key={id}></div>

    })}
  </div>
);
};
